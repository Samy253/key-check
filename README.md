## Keyboard Event Detector

A simple front-end project built with HTML, CSS, and JavaScript.
The app detects any key you press and displays its Key, Key Code, and Code in a table on the screen.

### What I practiced

- DOM manipulation (getElementById, innerHTML)
- Listening to keyboard events with addEventListener
- Using event object properties (e.key, e.keyCode, e.code)
- Dynamic HTML generation using template literals

Check it out [here](https://github.com/Samy253/key-check)
